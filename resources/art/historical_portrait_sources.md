# Historical Portrait Sources

This file is the locked portrait manifest for leaders already implemented in the AI.
Selection priority is historical fit first, then licensing clarity, then image quality.

Use the `download_url` values directly. They are Commons `Special:Redirect/file/...` links, which are stable enough for scripted downloads and avoid hard-coding Wikimedia hash paths.

## Practical Rule

All entries above are acceptable for local mod packaging, but `valette.jpg` is the one outlier because it is share-alike rather than public domain or CC0. If the redistribution plan tightens later, replace Valette before touching the others.

## Candidate UI Hook

The most likely game-facing portrait hook discovered in this repo is in `data/civmods.xml`, under each civ's `MatchmakingTextures` block:

- `PortraitTexture`
- `PortraitTextureCoords`
- `SmallPortraitTexture`
- `SmallPortraitTextureCoords`

This appears to drive the civilization portrait shown in quick-launch or AI-selection style UI, and is the cleanest current candidate for wiring Legendary Leaders headshots. The remaining unknown is asset format and path convention: existing entries point at stock `ui\\...` textures and sprite sheets, so the staged historical files in `art/ui/leaders/` should be treated as source assets until an in-game test confirms whether they can be referenced directly or need conversion into a `ui\\...` texture asset.
