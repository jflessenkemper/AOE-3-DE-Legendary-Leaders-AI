# Legendary Leaders AI

Legendary Leaders AI is a standalone Age of Empires 3: Definitive Edition overhaul built around the full base-civilization roster plus the fully playable revolutions. This README is a direct reference for every current playable nation, its assigned commander, its AI playstyle, and the currently authored taunts.

Each nation entry now also lists its surrender tactic. These lines map to the AI prisoner doctrines currently wired in the mod, covering prison placement, guard posture, and captive-handling behavior while the full unit-side surrender command continues to expand.
