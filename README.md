# Legendary Leaders AI

**The ultimate historical AI overhaul for Age of Empires 3: Definitive Edition**

Legendary Leaders AI transforms every civilization in the game — including all revolutionary nations as fully playable options in the lobby — into a deeply researched, living historical commander.

Each leader features a multi-layered personality that governs:
- Unique base placement and expansion logic
- Leader-specific walling and defensive strategies
- Dynamic economies that evolve throughout the game
- Tailored military doctrines and battlefield tactics
- Naval behavior and economy booming
- Distinct personality quirks that feel truly authentic

**Features**
- Fully playable revolutionary nations directly in the Skirmish and Multiplayer lobby
- Historical leader portrait source set prepared for implemented leaders, with local assets staged under `art/ui/leaders/`
- Candidate civ portrait hookup identified in `data/civmods.xml` via `MatchmakingTextures`, pending asset-format validation
- Historical single-colour player palette
- In-character insult and compliment taunts
